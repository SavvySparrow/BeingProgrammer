A javascript statement with write Method


Write():Write some text to the output stream:


<script>

document.write("Hello World!");

</script>


Writeln():Same as write(), but adds a newline character after each statement


Example of difference between write() and writeln()


<p>Note that write() does NOT add a new line after each statement:</p>


<pre>

<script>

document.write("Hello World!"); document.write("Have a nice day!");

</script>

</pre>


<p>Note that writeln() add a new line after each statement:</p>


<pre>

<script>

document.writeln("Hello World!"); document.writeln("Have a nice day!");

</script>

</pre>
